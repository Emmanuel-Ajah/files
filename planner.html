How to use the Planner

✍️

1. Add a Task

Type your task description, pick a category and priority, then hit Add.

☑️

2. Mark Complete

Click the checkbox on any task to mark it as done. Click again to undo.

🔍

3. Filter Tasks

Use the filter buttons to view tasks by status or category.

🗑️

4. Delete Tasks

Hit the ✕ button to permanently remove a task from the list.

💾 Tasks are saved automatically in your browser's localStorage — they persist between visits.
